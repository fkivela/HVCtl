Installation¶

Dependencies¶

HVCtl was written and tested in Python 3.7.3, so that or another relatively new Python version should be installed.

In addition to the Python standard library, HVCtl uses pySerial (tested with version 3.4) and urwid (tested with version 2.0.1). pySerial must be installed for HVCtl to work, but HVCtl can run without urwid by using a simpler user interface (see Usage for details).

The HVCtl directory¶

HVCtl doesn’t include an installation script; simply download the HVCtl directory to a location of your choosing. In order to access the hvctl package and the hvctl-run script, the HVCtl directory should be added to $PYTHONPATH and $PATH or made the working directory.
