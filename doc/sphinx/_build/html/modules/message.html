message¶

This module defines the Message class for communicating with the HV generator.

hvctl.message.COMMANDS = ['set voltage', 'set current', 'get voltage', 'get current', 'hv on', 'hv off', 'set mode', 'set inhibit', 'get status']¶

A list of valid commands.

class hvctl.message.Message(command, value=None, is_answer=False)¶

This class represents a message sent to or received from the HV generator.

command¶

The command that is sent to the HV generator, or the command to which self is an answer; e.g. 'set voltage' or 'hv on'. All valid commands are listed in COMMANDS.

Type

string

value¶

A value associated with the command, e.g. the value to which the voltage should be set. This should be left to None when sending getter or status commands. For current and voltage, value ranges between 0 and INT_MAX; the actual voltage or current value in V or mA can be computed by multiplying value with DELTA_U or DELTA_I.

Type

int or None

is_answer¶

Determines whether self represents a message to the HV generator or a response from it.

Type

bool
