virtualhv¶

This module defines the VirtualHV class.

class virtualhv.VirtualHV(*args)¶

Bases: virtualconnection.VirtualConnection

A virtual high voltage generator.

This class simulates a HV PSU that can be communicated with through a serial connection. This makes it possible to test HVCtl without access to a physical device.

The voltage of the virtual HV PSU in V.

HV_off(value)¶

Handle a 'HV off' command.

If HV_off_command is False and value is True, HV_off_command is set to True.

If HV_off_command is True and value is False, HV_on_status is set to False and HV_off_command is set to False.

get_status(_)¶

Handle a 'get status' command.

This method returns an 8-bit integer describing the current status of the virtual HV PSU (excluding voltage and current). The values of the bits in the number are copied from the attributes of self in the following manner:

Bit 0

inhibit

Bit 1

mode (1 for 'local', 0 for 'remote'),

Bit 2

HV_off_command,

Bit 3

HV_on_command,

Bit 4

HV_on_status,

Bit 5

interlock,

Bit 6

fault,

Bit 7

regulation (1 for 'voltage', 0 for 'current'),

Since True equals 1 in Python, attributes with a value of True will set their respective bit to 1, and attributes with a value of False will set their bit to 0. The bits are in big-endian order.

Returns

An int in range(256).
