advanced_tui¶

This module defines the AdvancedTUI class.

This module uses urwid. If urwid isn’t installed, command_line_ui.CommandLineUI provides a UI that can be run without it.

class advanced_tui.AdvancedTUI(script, inputfile, outputfile)¶

A text-based user interface combining a terminal-style command line interface and a text screen.

display¶

A text field for presenting information to the user. Use advanced_tui.display.set_text() to change its contents.

Type

urwid.Text

cli¶

A command-line interface for issuing commands; located below the display.

Type

CLI

__init__(script, inputfile, outputfile)¶

Initialize a new AdvancedTUI.

Parameters

  • script – The program controlled by the TUI. The program should start when script() is called.

  • inputfile – A file-like object where user input is sent.

  • outputfile – A file-like object where script should print its output, if it is to be printed on the screen.

keypress(size, key)¶

Handle key presses.

Pressing ‘q’ after script() has returned exits the program.

run()¶

Start script and the UI loop in parallel threads.
