advanced_tui¶

This module defines the AdvancedTUI class.

This module uses urwid. If urwid isn’t installed, CommandLineUI provides a UI that can be run without it.

class hvctl.advanced_tui.AdvancedTUI(script, inputfile, outputfile)¶

A text-based user interface combining a terminal-style command line interface and a text screen.

display¶

A text field for presenting information to the user. Use display.set_text to change its contents.

Type

urwid.Text

cli¶

A command-line interface for issuing commands; located below the display.

Type

CLI

__init__(script, inputfile, outputfile)¶

Initialize a new AdvancedTUI.

Parameters

  • script – An object that should run the program controlled by the TUI when called with script().

  • inputfile – A file-like object where user input is sent.

  • outputfile – A file-like object where script should print its output, if it is to be printed on the screen.

keypress(size, key)¶

Handle key presses.

Pressing ‘q’ after the program has returned exits the UI.

run()¶

Start the program and the UI loop in parallel threads.
