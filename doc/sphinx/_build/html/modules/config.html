config¶

This module defines several constants based on the options defined in a configuration file. The configuration file should be named 'hv.conf' and be located in the same directory as this module.

hvctl.config.SERIAL_KWARGS¶

A dict of keyword arguments and values for creating a serial.Serial object. The following keyword arguments and their values are included: port, baudrate, bytesize, parity, stopbits, timeout, xonxoff, rtscts, dsrdtr. The exclusive keyword will also be included and set to True, if the version of the serial module is 3.3 or later. See the documentation of serial.Serial for further details.

hvctl.config.MAX_VOLTAGE¶

The highest voltage the HV PSU can produce in V. For negative polarity devices, the value is negative. The value is used for scaling the 12-bit integer used by the Technix serial protocol for setting the voltage.

hvctl.config.VOLTAGE_LIMIT¶

The maximum voltage for a particular application in V. HVCtl will never set the voltage to a value exceeding this. For negative polarity devices the value should be negative.

hvctl.config.MAX_CURRENT¶

The highest current the HV PSU can produce in mA. The value is used for scaling the 12-bit integer used by the Technix serial protocol for setting the current.

hvctl.config.CURRENT_LIMT¶

The maximum current for a particular application in mA. HVCtl will never set the current to a value exceeding this.

hvctl.config.INT_MAX¶

The serial protocol uses 12-bit unsigned integers for the values of voltage and current. This is the maximum value of that integer, 4095.

hvctl.config.DELTA_U¶

The smallest amount by which the value of the voltage can be changed, in V.

hvctl.config.DELTA_I¶

The smallest amount by which the value of the current can be changed, in mA.

hvctl.config.main()¶

Parse the configuration file and define the constants. This function is called automatically when the module is imported.

Raises

  • FileNotFoundError – If the configuration file cannot be found.

  • RuntimeError – If the configuration file is found but cannot be parsed.
