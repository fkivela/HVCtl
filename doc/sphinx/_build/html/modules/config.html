config¶

This module defines several constants based on the options defined in a configuration file. The configuration file should be named 'hv.conf' and be located in the same directory as this module.

config.SERIAL_KWARGS¶

A dict of keyword arguments and values for creating a serial.Serial object.

config.VOLTAGE_LIMIT¶

The maximum value allowed for the voltage, in V.

config.CURRENT_LIMT¶

The maximum value allowed for the current, in mA.

config.INT_MAX¶

The serial protocol uses 12-bit unsigned integers for the values of voltage and current. This is the maximum value of that integer, 4095.

config.DELTA_U¶

The smallest amount by which the value of the voltage can be changed, in V.

config.DELTA_I¶

The smallest amount by which the value of the current can be changed, in mA.

config.main()¶

Parse the configuration file and define the constants. This function is called automatically when the module is imported.

Raises

  • FileNotFoundError – If the configuration file cannot be found.

  • RuntimeError – If the configuration file is found but cannot be parsed.
