command_line_ui¶

This module defines a simple command-line UI for HVCtl.

class command_line_ui.Command(names, function, args, description)¶

A namedtuple to represent accepted commands.

names¶

A list of aliases the command can be called with.

Type

list of strings

function¶

The name of the method called with the command.

Type

string

args¶

A list of arguments accepted by the command. E.g. [‘x’, ‘y’, ‘z=None’].

Type

list of strings

description¶

A description shown to the user.

class command_line_ui.CommandLineUI(*args, **kwargs)¶

A simple command-line UI.

stopflag¶

A True value of this flag tells the UI loop to stop.

Type

bool

__init__(*args, **kwargs)¶

Initialize a new CommandLineUI.

Args and kwargs are passed to API.__init__().

commandlist¶

Class attribute. A list of accepted commands (as Command objects).

commands¶

Class attribute. A dict of accepted commands, with command names as keys and Command objects as values. Aliases for the same command are represented with different keys mapping to the same value.

halt(*args, **kwargs)¶

Call API.halt() and stop the UI by setting stopflag to True.

help()¶

Print a help message.

input(prompt='')¶

Ask the user for input.

Works like the built-in input() function.

print(*objects, sep=' ', end='\n')¶

Print text to the UI.

Works like the built-in print() function, but without the file argument.

prompt = '>> '¶

Class attribute. The prompt displayed before waiting for input.

run()¶

Start the UI.

run_command(line)¶

Parse line and run it as a command.

start_msg = "Welcome to HVCtl! Type 'help' for a list of commands."¶

Class attribute. The message displayed when the UI is started.

stop()¶

Stop the UI by setting stopflag to True.
