widgets¶

This module uses the urwid library to create a TUI that contains a scrollable command line interface and a screen above that.

class hvctl.widgets.CommandHistory¶

A class to store command history.

This class emulates the command history browsing behaviour of a standard Linux terminal.

history¶

A list of previously entered commands. Index 0 is filled with None to give history and temp_history the same length.

temp_history¶

The same as history, but includes the newest command (the one that is currently being written) at index 0. If any previously given commands are modified (through update_command()), the modified forms are saved here, while history contains the original forms.

index¶

The index of the currently selected command. 0 is the command currently being written; 1 is the previous one etc.

__init__()¶

Initialize history to [], temp_history to [None] and index to 0.

down()¶

Decrease index by 1 (i.e. select a command that is newer than the current one).

If the current command is the newest one, nothing is done.

enter_command()¶

Enter the currently selected command.

This saves the command into memory and and resets index to 0. If the command was created by modifying a previous command, that command’s history entry will be reset into its original form. If multiple identical commands are entered consecutively, only one is saved, and empty commands are not saved at all.

get_command()¶

Return the currently selected command as a string.

up()¶

Increase index by 1 (i.e. select a command that is older than the current one).

If the current command is the oldest one, nothing is done.

update_command(string)¶

Change the currently selected command into string.

class hvctl.widgets.CommandLines(inputfile, outputfile)¶

This widget provides a terminal-like command line interface.

Entering commands scrolls the screen upwards so that the current command is always visible on the screen, but the functionality to scroll up to view old commands is provided by the ScrollableWidget class instead of this class.

This is a box widget, so its container sets its height and width.

input_queue¶

Input entered to self by the user is saved here, and can be read by a program.

Type

queue.Queue

output_queue¶

Output enetered here is printed to self.

Type

queue.Queue

history¶

An object that stores previously given commands.

Type

CommandHistory

edit¶

A widget that stores the text displayed by self and provides functionality for writing input.

Type

urwid.Edit

__init__(inputfile, outputfile)¶

Initialize a new CommandLines object.

Parameters

process_command – The method that should be used for processing the commands given to the TUI. The method should accept a string as an argument, perform whatever tasks the command requests, and return any possible output as a string. If the command produces no output, the method should return None instead.

enter()¶

Enter the current editable string as a command.

history_down()¶

Scroll command history down one step.

history_up()¶

Scroll command history up one step.

keypress(size, key)¶

Handle key presses.

The up and down arrow keys are used to browse command history. Enter enters a command.

move_cursor_to_end()¶

Move the cursor to the end of the editable string.

update()¶

Print the contents of output_queue to the screen. If output_queue is empty, do nothing.

class hvctl.widgets.Position(listeners)¶

A class to keep track of the position to which a terminal is scrolled.

listeners¶

The widgets that should be re-rendered whenever self is changed.

Type

iterable

total_rows¶

The number of rows in the terminal, counting also those that are not currently visible on the screen. Empty rows are included. Whenever the terminal is re-rendered, the terminal widget should update this attribute.

Type

int

visible_rows¶

Like total_rows, but only counts the rows visible on the screen.

Type

int

relative¶

A relative scroll position between 0 (scrolled to the top) and 1 (scrolled to the bottom). Setting this also changes the value of absolute.

Type

float

absolute¶

An absolute scroll position given as the number of invisible rows that are located above the top edge of the terminal.

Type

int

__init__(listeners)¶

Initialize a new Position object.

Parameters

listeners (iterable) – The initial value for listeners.

property absolute

Get or set absolute. Values below 0 are set to 0 and values over max_absolute to max_absolute.

property max_absolute¶

The maximum value of absolute, equal to total_rows - visible_rows.

property relative

Get or set relative. Values below 0 are set to 0 and values over 1 to 1.
